Cryptography

STEP 1 : Consider set of equations of the form ; X= a1 mod m1 ,X= a2 mod m2 ….,X=an mod mn.

STEP 2 :: Find M using M=m1 x m2 x …. x mn.This is the common modulus.

STEP 3 :: Find M1 = M/m1 , M2 = M/m2 , … , Mn = M/mn.

STEP 4 :: Find the multiplicative inverse of M1,M2…,Mn using corresponding moduli (m1,m2,...mn). Call the inverse as M1^-1,M2^-1,...,Mn^-1.

STEP 5 :: Now the final solution to the simultaneous equations is

X= ( a1 x M1 x M1^-1 + a2 x M2 x M1^-2 +.... + an x Mn x Mn^-1 ) mod M
